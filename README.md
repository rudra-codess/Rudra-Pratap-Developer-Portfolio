## ⚙️ Installation

Clone the repository

```
git clone https://github.com/rudra-codess/Rudra-Pratap-Developer-Portfolio.git
```

Go to project folder

```
cd Rudra-Pratap-Developer-Portfolio
```

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```

Open browser

```
http://localhost:5173
```
